For a Mac:  
	(1)  Install homebrew (a package manager):

>> /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

	(2)  Install python3 by running:   brew install python3
	(3)  run using     python3 hello.py

For Linux:
	(1)  in the terminal, run:  sudo apt-get install python3
	(2)  also run			sudo apt-get install pip3
	(3)  run using	python3 hello.py

For a PC:   
	(1)  Install Chocolatey (a package manager)

>> Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

	(2)  Close and reopen the command prompt	
	(3)  Install python3 using:     choco install -y python 
	(4)  run using       	    python hello.py

If you know of and prefer some alternative for running python code, you are free to use it.
